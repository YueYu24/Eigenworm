# Eigenworm
This repository uses worm chemotaxis data from the Barrios lab to practice data analysis and figure making. 

Original data can be found at https://github.com/Barrios-Lab/Chemotaxis-Data-and-Analysis. 

The corresponding paper is "Conflict during learning reconfigures the neural representation of positive valence and approach behavior", https://doi.org/10.1016/j.cub.2024.10.024.
